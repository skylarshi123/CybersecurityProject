Azure Fundamentals AZ-900
This certification will introduce you to concepts on Azure Fundamentals. If you are in the field a popular path is Azure Administrator then Azure Solutions Architect. People say to skip Azure Fundamentals, but I would say it is good to build momentum. There is exam overlap and this exam is free for all with the virtual training voucher. If you don't work in the field and just want to learn, Azure is great for projects so this certification will allow you see what technology is out there.
