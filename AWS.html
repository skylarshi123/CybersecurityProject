AWS Cloud Practitioner
This certification will introduce you to concepts on AWS Fundamentals. If you are in the field a popular path is the associate solutions architect and go from there. People say to skip AWS cloud practitioner, but I would say it is good to build momentum. There is exam overlap but unlike Azure this exam is not free. If you don't work in the field and just want to learn, AWS is great for projects so this certification will allow you see what technology is out there. AWS is leading in the industry for the cloud.
